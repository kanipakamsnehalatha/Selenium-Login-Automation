
# Selenium Login Automation Project

This project demonstrates automated testing of login functionality using Selenium WebDriver with Python.

## Technologies Used
- Python
- Selenium WebDriver
- WebDriver Manager

## Features
- Automates login form submission
- Uses ID and CSS selectors
- Implements Explicit Waits
- Validates successful login message

## How to Run
1. Install Python
2. Install dependencies:
   pip install selenium webdriver-manager
3. Run:
   python test_login.py
